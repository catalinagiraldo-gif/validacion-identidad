# Prototype Index

Catalog of all wireframes generated by the RPP (Rapid Prototyping Pipeline).

## How to add a prototype

1. Create a branch: `wireframe/DROP-XXXX-feature-name`
2. Generate components with Claude Code using the DS Registry
3. Add the route to `src/app/app.routes.ts`
4. Document below
5. Open a PR using the template

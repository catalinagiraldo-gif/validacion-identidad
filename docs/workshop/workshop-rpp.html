Que es RPP Hub?

  • Portal privado en dropitesters.co
  • Prototipos interactivos en Angular
  • Dual architecture: antigua + nueva
  • 36 pantallas en nueva arquitectura
  • Login con Google @dropi.co

Stack Tecnologico

Angular 17.3Framework
SCSS + DS TokensEstilos
PrimeNGComponentes UI
Firebase AuthAutenticacion
VercelDeploy
Claude CodeAI Assistant

Prerrequisitos

  • Git instalado y configurado
  • GitHub Desktop (opcional, recomendado)
  • Node 22.7+ via nvm
  • Yarn como package manager
  • VS Code o Cursor
  • Claude Code Desktop
  • Cuenta GitHub con acceso al repo

Paso 1

Clonar el repositorio

git clone https://github.com/producto-rpplab/dropi-prototypes.git
cd dropi-prototypes

Paso 2

Instalar dependencias

# Activar Node version correcta
export NVM_DIR="$HOME/.nvm"
. "$NVM_DIR/nvm.sh"
nvm use

# Instalar paquetes
yarn install

Paso 3

Compilar y ejecutar

npx ng serve --open

# Abre automaticamente en:
# http://localhost:4200

Paso 4

Activar Claude Code

# Dentro del proyecto:
claude

# Claude lee CLAUDE.md automaticamente
# Conoce la arquitectura, tokens y convenciones

Paso 5

Documentacion clave

CLAUDE.md -- reglas del proyecto
navigation-map-new.json -- rutas
ds-registry/ -- Design System
src/app/pages/new/ -- 36 pantallas
wireframe-prompt-template.md -- template
